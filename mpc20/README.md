## 한국어

# MPC20 - 웹 오디오 샘플러

Yeezus 투어의 일부였던 시드니 공연 라이브 영상을 보고 아주 큰 감명을 받았다.
그렇게 만들게 된 간단한 샘플러

![Kanye West Sydney Concert](https://img.youtube.com/vi/uCMG1vayTnA/0.jpg)
[라이브 영상 보기](https://youtu.be/uCMG1vayTnA?si=Xnc8Ll8a1Swra9dr)

랩탑이나 데스크탑에서는 키보드, 마우스 둘 다 사용 가능하고 모바일에서는 터치하면 된다.

### 기능

- 16개의 패드로 사운드 재생
- 키보드 지원 (Z-V, A-F, Q-E, 1-4)
- 다중 사운드 레이어링

### 사용법

- **마우스/터치**: 패드를 클릭하거나 탭하여 사운드 재생
- **키보드**: Z, X, C, V, A, S, D, F, Q, W, E, R, 1, 2, 3, 4 키 사용
- **패드 16 (4 키)**: 모든 재생 중인 사운드 정지
- **INFO 버튼**: 프로젝트의 버전 확인, 작성자 인스타그램 링크, 소스코드 링크

### 오디오 시스템

Web Audio API를 사용하여:
- 지연시간 없는 사운드 재생
- 동시 다중 사운드 재생
- 모바일 자동재생 정책 준수

### 브라우저 지원

- Chrome/Edge
- Firefox
- Safari

### 라이선스

개인용으로만 사용 가능 (상업적 사용 금지)

## English

# MPC20 - Web Audio Sampler

I was deeply moved by the live video of the Sydney concert from the Yeezus tour.
This is a simple sampler I built to revisit that feeling from time to time.

![Kanye West Sydney Concert](https://img.youtube.com/vi/uCMG1vayTnA/0.jpg)
[Watch the live video](https://youtu.be/uCMG1vayTnA?si=Xnc8Ll8a1Swra9dr)

On laptop or desktop, you can use either keyboard or mouse. On mobile, use touch.

### Features

- 16 pads for sound playback
- Keyboard support (Z-V, A-F, Q-E, 1-4)
- Multi-sound layering

### Usage

- **Mouse/Touch**: Click or tap pads to play sounds
- **Keyboard**: Use Z, X, C, V, A, S, D, F, Q, W, E, R, 1, 2, 3, 4 keys
- **Pad 16 (4 key)**: Stop all currently playing sounds
- **INFO Button**: Check project version, author Instagram link, and source code link

### Audio System

Using Web Audio API:
- Low-latency sound playback
- Simultaneous multi-sound playback
- Mobile autoplay policy compliance

### Browser Support

- Chrome/Edge
- Firefox
- Safari

For personal use only (Commercial use prohibited)

## 日本語

# MPC20 - ウェブオーディオサンプラー

Yeezusツアーのシドニー公演のライブ映像を見て、とても感動した。
あの時の感覚を時々思い出すために作ったシンプルなサンプラーだ。

![Kanye West Sydney Concert](https://img.youtube.com/vi/uCMG1vayTnA/0.jpg)
[ライブ映像を見る](https://youtu.be/uCMG1vayTnA?si=Xnc8Ll8a1Swra9dr)

ラップトップやデスクトップではキーボードとマウスの両方が使え、モバイルではタッチで操作できる。

### 機能

- 16個のパッドでサウンド再生
- キーボードサポート (Z-V, A-F, Q-E, 1-4)
- マルチサウンドレイヤリング

### 使い方

- **マウス/タッチ**: パッドをクリックまたはタップしてサウンド再生
- **キーボード**: Z、X、C、V、A、S、D、F、Q、W、E、R、1、2、3、4キーを使用
- **パッド16 (4キー)**: 再生中のすべてのサウンドを停止
- **INFOボタン**: プロジェクトのバージョン、作者のInstagramリンク、ソースコードリンクを確認

### オーディオシステム

Web Audio APIを使用して:
- 低遅延のサウンド再生
- 複数のサウンドの同時再生
- モバイルオートプレイポリシーの順守

### ブラウザサポート

- Chrome/Edge
- Firefox
- Safari

### ライセンス

個人使用のみ (商業使用禁止)
